# Godot Launcher

Welcome to the official Godot Launcher documentation.

**Godot Launcher** is a standalone app to manage official releases and custom-built Godot editors, organize projects, and automate common project setup tasks so you can focus on building games.

## Features at a glance

- **Drag-and-drop project import**: Drop one or many `project.godot` files into the Projects list to add them quickly with visible progress.
- **Flexible project location**: Create a new project in your default Projects folder or temporarily override the path for one project.
- **Git and VS Code integration**: Initialize Git and configure VS Code for project workflows.
- **Project tool toggles**: Enable Visual Studio Code integration or initialize Git from any project overflow menu.
- **Official and custom editor management**: Install official Godot releases, register custom-built Godot editors, and choose the right editor for each project.
- **Custom-Built Godot Editor Support**: Register custom-built Godot editors and optionally create their launcher manifest from the app. See [Custom-Built Godot Editors](/guides/custom-editors).
- **Project settings**: Rename launcher project entries and optionally update the Godot project name in `project.godot`.
- **Per-project settings**: Keep editor configurations isolated to avoid conflicts across projects.
- **System tray access**: Quickly open projects from the system tray.
- **Multilingual interface**: Detect your OS language and switch between supported locales from settings.
- **Update control**: Check for launcher updates automatically, download manually, restart when ready, and opt into prerelease builds from Settings.
- **Cross-platform support**: Windows ready, with macOS and Linux support in progress.
- **Free and open source**: Forever free and community-driven.
